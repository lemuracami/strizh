Для настройки нового обмена данными с другим узлом или с другой программой выберите соответствующую команду из подменю Создать. Откроется помощник настройки нового обмена данными.

Для выполнения обмена данными вручную выберите одну из настроек в списке и выполните команду Выполнить сейчас. При этом сначала будут загружены новые данные из другой программы, а затем выполнена выгрузка в обратную сторону. Кроме того, с помощью команды Настройка - Выполнить с настройкой можно выполнить обмен данными, предварительно настроив сопоставление одинаковых данных. Такая процедура полезна в тех случаях, когда в обоих узлах (программах) по каким-то причинам были независимо друг от друга введены одни и те же данные. Для снижения вероятности таких ситуаций можно настроить частый обмен данными по расписанию.

С помощью команды Настройка - Автоматически по расписанию можно настроить обмен данными по заданному расписанию. В этом случае, в клиент-серверном режиме работы обмен данными будет выполняться в фоне, с помощью регламентного задания Выполнение обмена данными, а в файловом режиме дополнительно потребуется запустить отдельный сеанс для выполнения регламентных заданий.

Замена типовых правил конвертации объектов и правил регистрации объектов из внешнего файла

Для замены типовых правил конвертации объектов и правил регистрации объектов правилами из внешнего файла необходимо:

  • Открыть обмен данными, для которого требуется изменить правила.
  • Из командной панели формы обмена данными в подменю Параметры обмена данными, в зависимости от того, какие правила требуется изменить, выбрать команду Открыть правила конвертации объектов или Открыть правила регистрации объектов.
  • В форме правил установить значение переключателя в положение Правила из файла и по кнопке Загрузить правила выбрать файл правил на компьютере или в сетевой папке.
  • Обратить внимание, что загружены корректные правила, подходящие для используемых версий конфигурации-источника и конфигурации-приемника. 
  • Нажать на кнопку Записать и закрыть. Загруженные правила будут применены немедленно.
