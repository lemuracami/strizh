Общая форма отчетов подсистемы CRM


Общие действия отчетов

Действия выполняются при нажатии кнопок командной панели формы.

Выбор периода отчета

Некоторые отчеты не требуют настройки периода. Если период не заполнен, отчет формируется за все время работы (если алгоритм формирования отчета допускает это).

Отборы

На основной форме отчета размещены только самые основные (быстрые) отборы. Все отборы отчета доступны через его настройку.

Настройка отчета


Настройки отчета - это установленные отборы, выбранные показатели, группировки и сортировки, показатели потребностей. Они изменяются как через форму отчета (быстрые отборы), так и через специальную форму настройки отчета.

Настройки отчета делятся на две группы:

  • созданные уполномоченным сотрудником коммерческой службы для всех пользователей;
  • созданные самим пользователем для себя.


Для сохранения текущей настройки используется форма сохранения настройки.

Действия над контрагентами, попавшими в результат отчета


Рассмотрим пункты меню подробнее:

  • Открыть - открывает значение, содержащееся в ячейке;
  • Контрагент - открывает контрагента, содержащегося в строке отчета;
  • Карточка клиента - формирует отчет "Карточка клиента" по контрагенту строки отчета;
  • Ход переговоров - формирует отчет "Ход переговоров" по контрагенту строки отчета;
  • Создать замечание - создает по контрагенту строки отчета событие с видом "Замечание";
  • Передать контрагента - создает по контрагенту строки отчета событие с видом "Передача".
