Регистр сведений "Доступность транспорта"

Форма редактирования доступности транспорта предназначена для интерактивного редактирования периодов доступности транспортных средств. Регистр вызывается для ввода и редактирования информации из пункта меню «Общие» / «Транспорт» / «Доступность транспорта».

При двойном клике по строке доступности открывается форма ввода нового периода доступности.
В окне ввода нового периода доступности транспорта необходимо выбрать транспортное средство, период, а так же значение доступности данного транспорта в данный период.
В реквизите "Гараж" необходимо указать местонахождение гаража.

!  Для рейсов местной доставки первый рейс в день будет рассчитываться из гаража.
