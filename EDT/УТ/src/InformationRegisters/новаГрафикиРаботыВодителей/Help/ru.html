Регистр сведений "Графики работы водителей"

Графики работы водителей предназначены для ввода периодов работы водителей и закрепление за водителем графика работы на выбранный период. Регистр вызывается для ввода и редактирования информации из пункта меню «Общие» / «Транспорт» / «Графики работы водителей»
 
Ввести новый период можно двойным кликом по строке графика или по кнопке "Ввести период работы".
 
Для ввода нового периода необходимо указать водителя, транспорт на котором он будет работает в этот период, период действия графика и график работы (по настройке графиков см. Графики работы).
Графики работы могут быть любой цикличности (через день, день через два и т.д.) однако для графиков предусматривающих еженедельный цикл необходимо, что бы начало действия графика приходилось на понедельник. Если день начала действия недельного графика отличается от понедельника, то будет выведено предупреждение.

 Для отображения графика доступны два представления:

  • По транспорту
  • По водителям
