Отчет "Структура доставки и посещения"

Отчет "Структура доставки и посещения" отображает сводную информацию по структуре следующих показателей:

  • Сумма заказов
  • Объем заказов
  • Вес заказов
  • Количество заказов
  • Количество точек доставки
  • Количество клиентов

Данные показатели доступны для анализа в разрезе аналитики (группировки):

  • По точке отправления
  • По точке прибытия
  • По транспорту
  • По водителю

Для анализа выводится сводная таблица и столбчатые диаграммы с накоплением.
