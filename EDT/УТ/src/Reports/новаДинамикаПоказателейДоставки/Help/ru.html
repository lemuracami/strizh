Отчет "Динамика показателей доставки"

Отчет "Динамика показателей доставки" отображает сводную информацию по динамике следующих показателей:

  • Сумма заказов
  • Объем заказов
  • Вес заказов
  • Количество заказов
  • Количество точек доставки
  • Количество клиентов

Данные показатели доступны для анализа по периодичности: день, неделя, месяц. Для анализа выводится сводная таблица и график динамики показателей.
