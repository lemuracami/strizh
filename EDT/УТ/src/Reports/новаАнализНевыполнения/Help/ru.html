Отчет "Анализ невыполнения"

Отчет "Анализ невыполнения" отображает сводную информацию по причинам невыполнения доставки. Для анализа доступны показатели:

  • Сумма заказов
  • Объем заказов
  • Вес заказов
  • Количество заказов
  • Количество точек доставки
  • Количество клиентов

Данные показатели доступны для анализа в разрезе аналитики (группировки):

  • По точке отправления
  • По точке прибытия
  • По транспорту
  • По водителю

Для анализа выводится сводная таблица и гистограмма.
