Планирование местной доставки

Форма "Планирование местной доставки" предназначена для ручного планирования рейсов доставки. Для планирования необходимо выбрать дату планирования и смену планирования. На основании этих данных будет отображаться информация в окнах "Доступный транспорт", "Доставки", "Запланированные рейсы".

Раздел "Доступный транспорт"

Раздел доступный транспорт содержит информацию о периодах доступности транспорта на дату планирования (с учетом выбранной смены планирования). Таблица содержит пересечение доступности транспорта и графиков работы водителей с учетом уже запланированных рейсов.

Если на текущую дату было произведено оперативное планирование (кнопка "Назначить районы"), то в разделе "Доступный транспорт" отображается распределение транспорта по районам указанным в форме оперативного планирования.

Раздел имеет предопределенные способы отображения:

  • Список - доступность транспорта выводится списком без группировок;
  • По району - доступность выводится сгруппированная по гео. районам. Для группировки используется районы водителей или транспорта в зависимости от настройки
  • По началу рейса - доступность выводится сгруппированная по началу возможных рейсов (имеет смысл если гаражи транспорта находятся в разных местах)

Раздел "Доставки"

Раздел "Доставки" содержит информацию о согласованных доставках доступных для планирования на текущую дату планирования (с учетом выбранной смены планирования).

Раздел имеет предопределенные способы отображения:

  • Доставки - доставки выводятся списком без группировок;
  • По районам - доставки выводятся сгруппированные по гео. районам;
  • Доставки списком - расширенная форма вывода доставок списком.

Справа от названия раздела выводится суммарная информация по всем выделенным в данный момент срокам (или по группировке, если курсов установлен на строке группировки)

Если на текущую дату планирования остались незапланированные доставки, то выводится кнопка "N старых!", где N - количество доставок незапланированных с прошлых периодов. При нажатии на кнопку, откроется форма переноса доставок.

Раздел "Запланированные рейсы"

Раздел "Запланированные рейсы" содержит информацию о рейсах, по которым ведется планирование на дату планирования (с учетом выбранной смены планирования). Рейсы отображаются в виде дерева: Рейс - Адрес точки маршрута - Доставка. Если несколько добавлено несколько доставок в один адрес, то такие доставки будут сгруппированы.

Если кнопка "Отображать рейс на карте" - включена, то текущий выделенный рейс (а так же рейсы отмеченные рейсы) будут выведены на карту.

Что бы завершить плнирование рейсов, необходимо отметить нужные рейсы и нажать кнопку "Завершить планирование" - при этом выбранные рейсы перейдут к следующей стадии выполнения ("Согласование рейсов" или "Подготовка рейса" в зависимости от настроек).
