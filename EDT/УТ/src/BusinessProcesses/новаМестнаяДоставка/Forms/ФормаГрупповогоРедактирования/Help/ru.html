Местная доставка

Бизнес-процесс "Местная доставка" предназначен для хранения информации и контроля прохождения процесса доставки сборных грузов в пределах региона. Предполагается, что такие доставки осуществляются в пределах суток.

Раздел "Грузоотправитель"

В разделе "Грузоотправитель" указываются место загрузки и принадлежность (собственный склад или контрагент). Если установлен флаг "Склад", то выбор грузоотправителя производится из справочника "Склады" иначе из справочника "Контрагенты".

В реквизите "Точка" указывается местонахождение пункта загрузки. Точка выбирается из справочника "Точки доставки".

"Отправление с" ... "по" - указывается период в который должна произойти погрузка. Если флаг после слова "по" установлен, то дата окончания периода для погрузки будет доступна для редактирования (период погрузки может быть несколько дней).

"Длительность погрузки" - время, которое необходимо для погрузки.

Раздел "Грузополучатель"

В разделе "Грузополучатель" указываются место разгрузки и принадлежность (собственный склад или контрагент). Если установлен флаг "Склад", то выбор грузополучателя производится из справочника "Склады" иначе из справочника "Контрагенты".

В реквизите "Точка" указывается местонахождение пункта разгрузки. Точка выбирается из справочника "Точки доставки".

"Прибытие с" ... "по" - указывается период в который должна произойти разгрузка в точке доставки. Если флаг после слова "по" установлен, то дата окончания периода для разгрузки будет доступна для редактирования (период разгрузки может быть несколько дней).

"Длительность разгрузки" - время, которое необходимо для разгрузки.

Закладка "Груз"

На закладке "Груз" указываются сведению о грузе, его характеристиках и условиях перевозки.

"Стоимость" - указывается стоимость грузка в выбранной валюте.

"Вес", "Объем" - суммарные объемо-весовые характеристики груза в килограммах и куб.м. соответственно.

"Требуется к кузове" - количество паллет, если груз расположен на паллетах.

"Описание" - текстовое описание перевозимого груза.

Закладка "Спецификация груза"

На закладке "Спецификация груза" можно (но не обязательно) указать состав перевозимого груза в разрезе номенклатуры. Если табличная часть заполнена, то суммарный вес и объем доставки будет рассчитан автоматически по данным табличной части.

Закладка "Категории транспорта"

На закладке "Категории транспорта" указываются те категории транспорта, которые могут доставлять данный груз. Для определения категорий необходимо отметить флагами необходимые категории.
