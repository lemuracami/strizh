ИМПОРТ

Импорт выполняется по команде кнопки «выполнить импорт». Кнопка вызывается из формы планы импорта данных. Требуется выбрать файл импорта и выполнить команду «импортировать». Сформируется проверочная форма для импорта. На закладках будут указаны строки с объектами импорта. Синим цветом шрифта будет указаны строки найденные в 1С, зеленым – новые строки. После проверки, данные записываются по команде с формы «записать».
