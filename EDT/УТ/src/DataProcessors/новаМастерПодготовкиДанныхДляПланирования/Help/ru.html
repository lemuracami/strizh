Обработка "Мастер подготовки данных для планирования"

Обработка предназначена для подготовки данных необходимых для дальнейшего планирования. Обработка вызывается из пункта меню «Местная доставка» / «Планирование» / «Мастер подготовки данных для планирования».

При открытии обработки табличная часть заполняется списком точек доставки у которых не установлен флаг "Подготовлена". При активации строки таблицы, если точка геокодирована и при наличии подключенных карт, на карте будет отображен выбранный адрес.

При нажатии на кнопку "Завершить подготовку", все геокодированные точки у которых указано окно времени работы, будут автоматически помечены как подготовленные.

При нажатии на кнопку "Отменить доставки" все доставки находящиеся на планировании в данную точку будут отменены.

Поиск адреса

Перед началом использования в системе новых точек доставки, их необходимо геокодировать (определить географические координаты местонахождения).

Что бы найти адрес в адресной базе нажмите кнопку "Поиск адреса".

Что бы запустить автоматическое геокодирование нажмите кнопку "Геокодировать все точки".

Для настройки автоматического геокодирования доступны настроки (по кнопке "Настройка"):

  • При отсутствии искомого дома, выдать ближайшие дома - означает, что если запрашивается дом, которого нет в базе, то система вернет географические координаты ближайшего дома по той же стороне улицы;
  • Если корпус не указан, то брать минимальный из существующих - означает, что если в адресе не указан номер корпуса, а в адресной базе присутвуют адреса только с номерами корпусов этого дома, то система вернет адрес с минимальным корпусом.
