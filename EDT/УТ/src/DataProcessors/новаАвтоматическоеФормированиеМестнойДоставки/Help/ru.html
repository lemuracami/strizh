Обработка "Автоматическое формирование местной доставки"

Обработка "Автоматическое формирование местной доставки" предназначена для автоматического формирования бизнес-процессов местной доставки по настроенному расписанию точек доставки. Обработка вызывается из пункта меню «Местная доставка»  / «Планирование» / «Автоматическое формирование местной доставки».  

Для формирования необходимо указать дату на которую будут сформированы доставки, способ определения точки грузоотправителя и грузополучателя:

  • Склад - точкой отправления/получения груза будет выбранный склад;
  • Склад по району - точкой отправления/получения груза будет склад, который будет автоматически определен по району точки доставки и обслуживаемым районам складов;
  • Контрагент - точкой отправления/получения груза будет точка доставки контрагента.

Обработка выберет из базы данных, те точки доставки для которых настроен режим посещения и создаст для такой точки бизнес-процесс местной доставки на указанную дату.
