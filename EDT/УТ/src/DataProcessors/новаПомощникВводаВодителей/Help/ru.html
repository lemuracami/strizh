Помощник ввода водителей

В системе принято разделять учет водителей в системе по принадлежности к контрагенту. Для этого можно указать

·         Собственный водитель

·         Частный перевозчик

·         Сотрудник перевозчика

 

Закладка «Основные»

 

Заполняется характеристики транспорта, паспортные данные, и данные по вод. Удостоверению.

 

Закладка «Дополнительно»

включает дополнительные характеристики кузова и транспорта.

Будьте внимательны, реквизиты по ширине, высоте, глубине, вместимости размещены на этой закладке. Однако, характеристики кузова являются важными при подборе транспортных средств и их желательно указывать всегда точно и верно.

 

Закладка «Документы и сканы»

Содержит информацию по введенным документам водителя. Возможно подкрепить отсканированные документы. На закладку автоматически переносятся данные по введенным ранее паспортным реквизитам и водительскому удостоверению.

 

Помощник по вводу предполагает ввод основных данных по водителю. Однако, основополагающими данными при подборе транспортных средств являются районы работы, график доступности(работы) водителя, график доступности транспорта.
