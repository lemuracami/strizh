Selecting the configuration file template

The form is intended for selecting the configuration file template to be used as a base of the technological log configuration file.

You can select a built-in file template (built-in templates a marked with a picture before the template name) or a custom file template.

To delete a custom template, click Delete in the templates list shortcut menu. Only user templates can be deleted.

Click OK to load the selected template to the configuration file editor.
