Saving the template of the technological log configuration file

Specify the following parameters of the configuration file template to be generated:

  • Template name - brief template name. This name will be displayed in the list of the Select template window.
  • Template details - template details to be displayed on the right part of the Select template window.
  • What paths to directories must be prompted for when the template is selected.

Press OK to number and save the template file. The template cannot be saved with the number that already exists.
