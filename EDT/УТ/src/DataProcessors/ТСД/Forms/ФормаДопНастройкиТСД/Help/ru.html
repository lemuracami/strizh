Настройки  дополнительных параметров

 Доступны настройки параметров устройства:

  • Тип связи - тип интерфеса, по которому происходит взаимодействие с ТСД
  • Адрес сервера (<IP или имя>:<порт>) - адрес сервера терминалов Mobile SMARTS
  • Прокси - прокси-сервер через который осуществляется взаимодействие с сервером терминалов Mobile SMARTS
  • Порт прокси - порт прокси сервера

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
