Справочник "Районы"

В справочнике "Районы" хранится информация о географических районах регионов. Классификатор районов используется при заполнении адресов доставки. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие»  / «Адреса» / «Районы».  

Справочник подчинен справочнику "Регионы".
