Справочник "Конфигурации карт"

В справочнике "Конфигурации карт" хранятся варианты возможных настроек отображения карт. Окно карты отображается справа в сворачивающемся окне и показывается в зависимости от текущего режима активного окна. Справочник вызывается для ввода и редактирования информации из формы настройки конфигурации "Управление транспортной логистикой" («Сервис» / «Настройка "Управление транспортной логистикой"»).

Раздел "Окна" - в разделе "Окна" указывается список окон карты. Каждое окно может содержать несколько карт. Если задано более одного окна, то переключение между окнами производится нажатием кнопок с названием окна.

Раздел "Карты" - в разделе "Карты" указывается список карт, которые будут показываться в текущем окне. При добавлении новой карты необходимо выбрать тип карты:

  • Ингит - карты производства фирмы ООО "Ингит" (должен быть установлен компонент GWX ActiveX версии не ниже 5). О лицензировании см. документацию.
  • CityGuide - карты производства фирмы ООО "МИТ" (должен быть установлен компонент ActiveX CityGuideServer). О Лицензировании см. документацию.

Дополнительные настройки карт "Ингит"

"Файлы карт" - список файлов карт, которые будут отображаться совместно в одном окне. В одно окно рекоммендуется помещать только одномасштабные карты соседних регионов. При этом графы дорог соседних регионов будут совмещены и появится возможность строить маршруты между картами.

Не рекомендуется помещать в одно окно несколько разномасштабных карт, т.к. построение маршрута будет не корректным.

"Ускоренная прорисовка" - включает/выключает более быструю прорисовку объектов карты, при этом возможны некоторые незначительные искажения.

"Сглаженная прорисовка" - включает/выключает сглаженную прорисовку объектов карты, при этом скорость прорисовки карты может незначительно замедлиться.

 Дополнительные настройки карт "City Guide"

"Файлы карт" - список файлов карт, которые будут отображаться совместно в одном окне.

Кнопка "Заполнить" - заполнить географическую рамку по умолчанию
