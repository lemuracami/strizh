Справочник "Виды транспорта"

В справочнике "Виды транспорта" хранятся виды транспортных средств используемых в компании. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Транспорт» / «Виды транспорта».

Например: легковой, грузовой и т.д.

Реквизит "Тип транспорта" указывает классификацию по следующим типам:

  • Легковой
  • Грузовой до 3.5 тонн
  • Грузовой свыше 3.5 тонн

Информация о типе транспортного средства будет использоваться при прокладке маршрута по карте.

Реквизиты "Грузоподъемность" и "Полезный объем" указывают на наиболее типичные для данного вида транспорта объемо-весовые характеристики транспортного средства.

Информация о видах транспортных средств используется в справочнике "Транспорт".
