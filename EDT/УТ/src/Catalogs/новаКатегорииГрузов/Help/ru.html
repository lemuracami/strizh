Справочник "Категории грузов"

Справочник "Категории грузов" используется для указания произвольных категорий грузов в доставке. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Номенклатура» / «Категории грузов».

В табличной части "Исключаемые категории" можно указать зависимые категории, которые будут автоматически отключены при выборе данной категории.
