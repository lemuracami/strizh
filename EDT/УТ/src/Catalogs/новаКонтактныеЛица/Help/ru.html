Справочник "Контактные лица"

В справочнике "Контактные лица" хранится информация о контактных лицах контрагентов. Справочник подчинен справочнику  "Контрагенты". Справочник вызывается для ввода и редактирования информации из формы справочника "Контрагенты".

Для каждого контактного лица можно заполнить список контактной информации (адрес, телефон, e-mail, web). При нажатии на кнопку "Основной", текущий элемент контактной информации становится основным по данному контактному лицу.
