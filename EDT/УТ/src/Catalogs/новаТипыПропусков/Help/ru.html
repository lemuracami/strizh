Справочник "Типы пропусков"

В справочнике "Типы пропусков" хранится информация о типах пропусков транспортных средств, при ограниченном проезде по городу. Справочник используется в карточке транспортного средства. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие»  / «Транспорт» / «Типы пропусков».
