Справочник "Номенклатура"

В справочнике "Номенклатура" хранится информация о номенклатуре используемой в конфигурации. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Номенклатура» / «Номенклатура».

Для каждой номенклатуры указывается код, наименование, полное наименование, ставка НДС.

На закладке "Единицы измерения" заполняется список возможных единиц измерения данной номенклатуры.

На закладке "Категории транспорта" можно указать какие категории транспорта могут доставлять данный товар.
