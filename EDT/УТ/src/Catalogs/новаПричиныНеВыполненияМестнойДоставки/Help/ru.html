Справочник "Причины невыполнения местной доставки"

В справочнике "Причины невыполнения местной доставки" хранится информация о классификации причин невыполнения доставок. Данный справочник используется при заполнении справочника "Результаты местной доставки". Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Классификаторы» / «Причины невыполнения местной доставки».
