Справочник "Виды документов водителей"

В справочнике "Виды документов" хранится информация о видах документов водителей и транспортных средств. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Транспорт» / «Виды документов водителей».

В данный справочнике содержатся два предопределенных вида документов: "Водительское удостоверение", "Паспорт технического средства".

Данный справочник используется для заполнения реквизитов документов водителей.
