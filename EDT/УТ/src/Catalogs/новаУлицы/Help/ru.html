Справочник "Улицы"

В справочнике "Улицы" хранится информация о улицах населенных пунктов. Классификатор улиц используется при заполнении адресов доставки. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие»  / «Адреса» / «Улицы».  

Справочник подчинен справочнику "Населенные пункты".
