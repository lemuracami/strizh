Справочник "Цветовое оформление рейсов"

Справочник "Цветовое оформление рейсов" используется для указания параметров стилевого оформления рейсов на карте. Справочник вызывается для ввода и редактирования информации из формы настройки конфигурации "Управление транспортной логистикой" («Сервис» / «Настройка "Управление транспортной логистикой"»).

В справочнике необходимо указать наименование стиля оформления и цвет рейса.
