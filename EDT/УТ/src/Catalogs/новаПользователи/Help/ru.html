Справочник "Пользователи"

В справочнике "Пользователи" хранится информация о пользователях системы "Управление транспортной логистикой". Справочник вызывается для редактирования информации из пункта меню «Сервис» / «Пользователи».

Запись в справочник добавляется при первом входе нового пользователя.

Для каждого пользователя указываются основные значения:

  • основная конфигурация карт (карты, на которые для данного пользователя есть лицензии);
  • основной классификатор районов (классификатор районов для отображения на карте и группировке заказов в списках);
  • основная схема оформления журналов;

На закладке "Точки отправления" указываются точки отправления заказов за которые отвечает данный пользователь. По этим точкам будет происходить отбор во всех списках отображающих заказы на доставку и рейсы.

На закладке "Районы" указываются районы за которые отвечает данный пользователь. По этим районам будет происходить отбор во всех списках отображающиих заказы на доставку и рейсы.

На закладке "Настройки" указывается часовой пояс (относительно мирового времени) текущего пользователя.

На закладке "Основные реквизиты" указываются значения реквизитов по умолчанию для ввода новых документов.
