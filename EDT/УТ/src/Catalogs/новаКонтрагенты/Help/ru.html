Справочник "Контрагенты"

В справочнике "Контрагенты" хранится информация о контрагентах компании. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Контрагенты».

На закладке "Основные" указывается основная информация о контрагенте: полное наименование, ИНН, КПП и категория контрагента.

На закладке "Точки доставки" указывается список точек доставки данного контрагента.

На закладке "Контакты" в табличной части "Контактные лица" указывается список контактных лиц контрагента. При нажатии на кнопку "Основной" текущее контактное лицо становится основным. Основные контактные лица подсвечиваются жирным шрифтом.

В табличной части "Контактная информация контрагента" указывается список контактной информации относящейся к контрагенту в целом. При нажатии на кнопку "Основной" текущая контактная информация становится основной для вида контактной информации. Основные контактные данные по каждому виду, подсвечиваются жирным шрифтом.

На закладке "Договоры" указывается список договоров с данным контрагентом.

На закладке "Банковские счета" указывается список банковских счетов контрагента.
