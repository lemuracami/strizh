Справочник "Контактная информация"

В справочнике "Контактная информация" хранится информация о вариантах географического деления карты на районы. Справочник подчинен справочникам "Водители", "Контрагенты", "Экспедиторы", "Контактные лица". Справочник вызывается для ввода и редактирования информации из форм справочников, которому принадлежит данная контактная информация.

Для каждого элемента контактной информации можно выбрать вид контактной информации (адрес, телефон, e-mail, web), указать значение и комментарий.
