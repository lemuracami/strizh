Справочник "Адреса"

В справочнике "Адреса" хранится информация о адресах доставки. Классификатор адресов используется при заполнении адресов торговых точек контрагентов и бизнес-процессов доставки.

Полный список адресов можно открыть из пункта меню «Общие» / «Адреса» / «Адреса». При выборе адреса в формах, где предусмотрен выбор адреса - открывается форма подбора адреса.

Если подключена карта CityGuide, то адресный классификатор можно обновить из карты. Для этого используйте обработку "Обновление адресного классификатора" ("Сервис" -> "Обновление адресного классификатора").
