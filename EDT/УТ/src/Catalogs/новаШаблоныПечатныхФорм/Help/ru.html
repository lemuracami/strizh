Справочник "Шаблоны печатных форм"

Справочник "Шаблоны печатных форм" предназначен для хранения настроек вывода печатных форм. Справочник вызывается для ввода и редактирования информации из формы настройки конфигурации "Управление транспортной логистикой" («Сервис» / «Настройка "Управление транспортной логистикой"»).

Существуют две предопределенных настройки:

  • Основная настройка пакетной печати - используется для настройки групповой печати сопроводительной документации по рейсу.

    В форме настроке флагами отмечаются те печтные формы, которые будут автоматически выводиться при нажатии на кнопку групповой печати из формы "Подготовка рейсов". В колонке "Количество" дополнительно указывается количество копий для печати. При установке флага "На принтер", печатная форма будет сразу же напечатана на принтере, без вывода на экран.

  • Основные печатные формы - используется для настройки отображения кнопок быстрого доступа к печтной форме из формы объекта, которой принадлежит эта печатная форма. В форме настроке флагами отмечаются те печтные формы, для которых в форме владельце печатной формы будет создана отдельная кнопка быстрой печати. В колонке "Количество" дополнительно указывается количество копий для печати. При установке флага "На принтер", печатная форма будет сразу же напечатана на принтере, без вывода на экран.
