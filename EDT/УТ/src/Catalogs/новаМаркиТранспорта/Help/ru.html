Справочник "Марки транспорта"

В справочнике "Марки транспорта" хранятся марки транспортных средств используемых в компании.

Например: МАЗ, ГАЗ и т.д.

Информация о марках транспортных средств используется в справочнике "Транспорт".

 

Для каждой марки транспорта возможен ввод норм расхода ГСМ.

 

В случае если требуется индивидуальный расчет по расходу ГСМ, для каждого транспортного средства вводятся значения потребления бензина/топлива подробнее см. описание справочник «транспорт».

 

Ввод нового значения осуществляется стандартными клавишами интерфейса программы 1с.

Дата начала действия – дата начала действия нормы. Определяет  границы действия норм.
 

Например:  Лето/ 20.06.2010, Осень/09.09.2010, Зима/09.11.2010, Весна/01.03.2010.

Для действия этих же норм следующего года требуется ввести строки

Лето/ 20.06.2011, Осень/09.09.2011, Зима/09.11.2011, Весна/01.03.2011.

 

ГСМ – Вид используемого ГСМ.  Используются значения из справочника «номенклатура».

Линейная норма на л/100км – норма расхода бензина/топлива на 100км

Норма на изменение собств. Веса л/100 тн.км. – количество литров используемого топлива при увеличении груза на 100 тонн на километр.

Норма на операцию, л. – количество литров для совершения операции на данной марке транспорта. Например, для уборочной машины, подъем мусорного бака.

Норма на ездку, л.  – количество литров требуемое на расход в рейсе.
