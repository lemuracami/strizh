Справочник "Результаты местной доставки"

Справочник "Результаты местной доставки" используется при заполнении отчета водителя. Справочник вызывается для ввода и редактирования информации из пункта меню «Общие» / «Классификаторы» / «Результаты местной доставки».

Реквизиты описывающие результат выполнения местной доставки непосредственно влияют на отчеты по статистике доставки.

Раздел "Отчет о выполнении"

Результат выполнения местной доставки - указывает, что данный результат выполнения необходимо считать как:

  • Выполнена - доставка считается выполненной полностью
  • Выполнена частично - произошли отклонения по грузу
  • Не выполнена - доставка не выполнена

Результат невыполнения местной доставки - указывает причину невыполнения (не имеет смысла если реквизит "Результат выполнения местной доставки" имеет значение "Выполнена".

Процент оплаты водителю - для разных результатов выполнения можно указывать различные проценты для оплаты доставки водителю. Может использоваться в случаях частичной доставки.

Точка посещена - указывает на посещение точки доставки водителем. Может использоваться для разграничения понятий "груз не доставлен" (например магазин был закрыт) и "водитель не заезжал" (например по причине ошибок планирования)
