Справочник "Начисления удержания"

Справочник "Начисления удержания" предназначен для указания статей начислений и удержаний для расчета зарплаты водителей. Справочник вызывается для ввода и редактирования информации из пункта меню «Местная доставка» / «Доходы и расходы» / «Регистрация доходов и расходов».

В элементе статьи начисления/удержания необходимо указать базу статьи: "Рейс", "Задание рейса" - указывает на объект на основании которого будет рассчитываться статья начисления.

Флаг "Удержание" устанавливается, если данная статья является статьей удержания.

Флаг "Рассчитываемое" устанавливается, если расчетной по формуле, при этом становится доступным поле "Формула расчета".

В формуле доступны ссылки на объект расчета:

  • Рейс - ссылка на бизнес-процесс "РейсМестнойДоставки"
  • ЗаданиеРейса - ссылка на бизнес-процесс "МестнаяДоставка"

Выбранные показатели доступны в формуле расчета по имени псевдонима %псевдоним%.
Результат вычислений должен быть присвоен переменной "Результат".

Например: Результат = ((Рейс.ДатаКонца - Рейс.ДатаНачала) / 3600)* %Показатель1%

Статью начисления/удержания необходимо сопоставить с статьей доходов/расходов (реквизит "Статья доходов расходов").
