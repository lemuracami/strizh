Справочник "Обмены данными"

Предназначен хранения настроек импорта / экспорта, используемых при обмене данными по расписанию а также в ручном режиме.

Основные реквизиты:

    • Каталог загрузки / Каталог выгрузки - путь к файлам для обмена
    • Флаг "Выполнять по расписанию" - при включении, появляется возможность выполнения обмена данными по расписанию (расписание можно задать кликнув по гиперссылке)
    • Флаг "Записывать лог" - при включенном флаге информация о загруженных объектах записывается в лог-файл (log.mxl) и помещается в архив (<каталог обмена>\архивы\ГГГГммДДЧЧММСС.zip)
    • Флаг "Выводить лог" - при включенном флаге информация о загруженных объектах выводится на экран в виде таблицы
    • Флаг "Использование семафора обмена" - возможность управления обменами с помощью файлов-семафоров:
      • Готовность, файл "ready.sem" - при видимости данного файла в каталоге обмена загрузка данных разрешается
      • Загрузка, файл "loading.sem" - в процессе загрузки ready.sem переименовывается в loading.sem
      • Ожидание файл "waiting.sem" - по окончании загрузки loading.sem  переименовывается в waiting.sem (состояние ожидания семафора готовности)

Табличная часть "Планы обмена" (только для типа обмена "Загрузка")

    • Задаются необходимые планы импорта, участвующие в загрузке.

Табличная часть "Имена файлов" (только для типа обмена "Загрузка")

    • В табличной части отображаются задачи, соответствующие текущему выбранному плану импорта. Для каждой задачи импорта задаются имена файлов (например: xx.dbf)
