Документ "Назначение схемы начисления зарплаты"

Документ "Назначение схемы начисления зарплаты" предназначен для указания конкретной схемы начисления каждому сотруднику. Документ вызывается для ввода и редактирования информации из пункта меню «Местная доставка» / «Зарплата» / «Назначение схемы начисления зарплаты».

Значение схемы для сотрудника начинает действовать с даты назначения (Реквизит "Дата назначения") до следующего переназначения.

В табличной части "Назначение схемы" необходимо выбрать сотрудника и соответствующую ему схему начисления.
