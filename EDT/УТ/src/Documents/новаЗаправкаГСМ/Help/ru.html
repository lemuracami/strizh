Документ «Заправка ГСМ»

Документ «Заправка ГСМ» предназначен для регистрации заправки транспортных средств.

Документ вызывается для ввода и редактирования информации из пункта меню «Общие» / «Учет ГСМ» / «Заправка ГСМ».
Альтернативно документ открывается на закладке «Путевой лист» отчета водителя.

В реквизитах документа указывается Наименование АЗС на которой совершалась заправка, транспортное средство, водитель осуществлявший заправку, количество, цена и дата заправки. Вид ГСМ и БАК выбираются автоматически по реквизитам карточки транспортного средства.

Для ввода очередной заправки возможно использовать один документ с добавлением строк в него либо создать новый документ.

Из формы «отчет водителя»:
• для ввода строки в существующий документ используется кнопка «добавить в существующий документ». из формы «отчет водителя» на закладке «путевой лист».
• для ввода нового документа используется кнопка «добавить в новый документ». из формы «отчет водителя» на закладке «путевой лист».

Из списка документов используя команду меню «Общие» / «Учет ГСМ» / «Заправка ГСМ».
