Документ «инвентаризация ГСМ»

Документ «инвентаризация ГСМ» предназначен для регистрации найденных отклонений в учете ГСМ.

Документ вызывается для ввода и редактирования информации из пункта меню «Общие» / «Учет ГСМ» / «Инвентаризация ГСМ в транспорте».

В документе указывается транспортное средство по которому будет проводиться инвентаризация. На основе количественного учета и ценовых показателей.
