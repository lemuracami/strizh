Документ "Заявка перевозчика"

Документ «Заявка перевозчика» предназначен для ввода заявки стороннего перевозчика о готовности предоставить транспорт. Список документов «Заявка перевозчика» вызывается для ввода и редактирования информации из пункта меню «Общие» / «Транспорт» / «Заявки перевозчиков».

В документе указывается информация о Водителе, Транспорте, Гараже, Контрагенте, Договоре, Контактном лице и направлении заявки, а так же указывается период действия заявки. После проведения заявки, указанный транспорт будет доступен для планирования наравне с собственным транспортом.
